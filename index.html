Introduction

C++ is a versatile, high-level programming language used for developing a wide range of applications. It combines the features of both procedural and object-oriented programming, making it a powerful and flexible choice for software development.

C++ provides a standard library that includes essential components like containers, algorithms, and input/output functions. It also offers core language elements such as data types, operators, control structures, and functions. C++ can be extended for various purposes through the creation of user-defined classes and objects.

This language can be employed in various contexts, including:

  • Application Development: C++ can be used to build desktop applications, games, and system software.
  • Embedded Systems: It's a popular choice for programming embedded systems due to its efficiency.
  • High-Performance Computing: C++ is well-suited for tasks that demand high computational speed.
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of computer programming concepts.
  • Familiarity with the concepts of variables, data types, and control flow.
  • Some prior experience with coding. If you're new to programming, consider starting with beginner tutorials.
C and C++

C++ shares similarities with the C programming language but offers additional features and a more robust standard library. C++ is an extension of C, with added support for object-oriented programming.

Hello World

To start writing C++ code, here's a basic "Hello World" program:

#include iostream
int main() {
std::cout << "Hello, World!" << std::endl;
return 0;
}

Compile and run this program to see "Hello, World!" printed to the console.

Variables

Variables in C++ are used to store and manipulate data. Variable names, called identifiers, must adhere to specific rules:

  • Must start with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can be letters, digits (0-9), underscores, or dollar signs.
  • C++ is case-sensitive, so "myVariable" and "myvariable" are distinct.

You can declare variables in several ways:

  1. Using the int keyword: int x = 42; (Declares and initializes an integer variable.)
  2. Without a keyword (not recommended for beginners): x = 42; (Creates a global variable.)
  3. Using the auto keyword (C++11 and later): auto y = 13; (Infers the data type.)
Data types

C++ provides several data types, including:

  • int: Integer data type.
  • double: Double-precision floating-point numbers.
  • char: Character data type
  • bool: Boolean data type.
  • string: String data type.
  • User-defined types (classes and structures)

These data types enable you to work with a wide range of values and perform various operations in your programs.

if...else statement

The if...else statement in C++ is used to control the flow of your program based on conditions. It allows you to execute different code blocks depending on whether a given condition is true or false.

if (condition) {
// Code to execute if the condition is true
} else {
// Code to execute if the condition is false
}

  • condition: An expression that evaluates to either true or false.
  • Code inside the first block (between { and }) executes if the condition is true.
  • Code inside the second block (after else) executes if the condition is false.

You can also use else if to handle multiple conditions in sequence:

if (condition1) {
// Code to execute if condition1 is true
} else if (condition2) {
// Code to execute if condition2 is true
} else {
// Code to execute if neither condition1 nor condition2 is true
}

while statement

The while statement in C++ is used to create loops that execute a block of code repeatedly as long as a specified condition remains true.

while (condition) {
// Code to execute while the condition is true
}

  • condition: An expression that determines whether the loop should continue.
  • The code block enclosed by { and } is executed as long as the condition is true.
  • If the condition becomes false, the loop terminates, and control moves to the next statement after the loop.
Function declaration

In C++, a function declaration is used to define a function's name, its return type, and the types of its parameters. It serves as a blueprint for the function's implementation. Here's the general syntax for declaring a function:

return_type function_name(parameter_type1 parameter1, parameter_type2 parameter2, ...) {
// Function implementation
}

  • return_type: Specifies the data type of the value that the function will return. Use void if the function doesn't return any value.
  • function_name: The unique name of the function.
  • parameter_type1, parameter_type2, etc.: The data types of the function's parameters.

Here's an example of a function declaration:

int add(int a, int b);

This declaration defines a function named add that takes two integer parameters (a and b) and returns an integer.
